Inicializa um problema linear diretamente a partir dos parâmetros fornecidos sem depender de arquivos.

Parameters
objective_functionArray representando o vetor da função objetivo.
constraint_matrixMatriz de restrições do problema.
is_maximizationBooleano que indica se o problema é de maximização.
restrictionsVetor das restrições.
restrictions_symbolsLista de símbolos das restrições (≤, =, ≥) (opcional, se não for passado, assumiremos que todos são ≤).

Configura diretamente as variáveis e a matriz de restrições, utilizando o mesmo algoritmo de base para executar o método Simplex. As variáveis são automaticamente nominais na forma x_1, x_2, ... para simplificar a visualização.
